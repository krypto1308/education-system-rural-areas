### Installation & Setup

#### Option 1: Direct File Access
1. Download all files to a folder
2. Open `index.html` in a web browser
3. The platform will work offline after first load

#### Option 2: Local Web Server
1. Install a local web server (e.g., Live Server, Python's http.server)
2. Serve the files from the project directory
3. Access via `http://localhost:port`

#### Option 3: Deploy to Web Hosting
1. Upload all files to a web hosting service
2. Ensure HTTPS is enabled for PWA functionality
3. Access via the provided URL
